# Health Clinic SaaS MVP

A minimal SaaS platform for clinics to manage patient visits and patient data.  
Designed for healthcare professionals to quickly book, view, and manage appointments.

## ✨ Features

- 🧑‍⚕️ Role-based view (doctors see all patients; patients see only their own)
- 📅 Schedule patient visits with time slots
- 📋 View scheduled visits in a dashboard
- 🔐 JWT-based login and password change
- 🧱 Multi-niche and organization support (Health/Fitness etc.)
- 📱 Fully responsive design for mobile

## 🛠️ Tech Stack

- Python 3.12
- Flask
- SQLite
- TailwindCSS via CDN (for styling)
- Hosted on Render (planned)
